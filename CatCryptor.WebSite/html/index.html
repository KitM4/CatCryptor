About this service_

[this service encrypts files of any format using the AES algorithm]

The CatCryptor web service provides a convenient and secure way to manage your files:

  • Upload your files and protect them with a password - only you can decrypt and view them.
  • When you need access to an encrypted file, just enter the correct password and the service will decrypt it for you.
  • Your information remains private and secure.

Easy to use: This web service is designed to be easy to use. Just select the files, set the password.
